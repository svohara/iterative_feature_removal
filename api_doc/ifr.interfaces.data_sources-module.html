load_flu_mat(matfile='Fluz_dat.mat', times=[14, 15, 16], id_filter=None)

source code 

Loads the Duke flu data, only those rows where the response is for the given time indexes.

Parameters:
  • matfile - A matlab file that contains duke influenza data, but converted to a format easily read by python/scipy. The .mat file should contain four matrices (not cell arrays), D, L, Time, ID, where D is the full data, L is the corresponding labels, Time is the time index of the data, and ID is the subject id for each row of data.
  • times - A list of time indexes that is used to filter the data. Only those rows corresponding to these time indexes will be included. Specify None to include all time steps.
  • id_filter - A list of ids to FILTER (remove) from the data. Specify None to include all subjects.
Returns:
tuple (D,L,Time,ID) as numpy arrays with the filtered data.

Note: For Duke Fluz data, use times=[14,15,16] and id_filter=None. For Duke H1N1 data, use times=[14,15,16] and id_filter = [1,3,5,10,18]
