Class CSV_data

source code

Class to encapsulate reading data from simple csv files where the first row is the column headers and fields are separated with commas. Rows are data samples, columns are field values.

Parameters:
  • datadir - The directory where the csv file is located
  • csvfile - The filename of the data file
  • label_cols - A list of the zero-indexed column numbers that should be treated as labels. These columns are assumed to have a discrete set of values, like a set of string labels, a set of integers. For regression data sets where the "labels" are continuous-valued numbers, set label_cols = None, and all the csv fields will be loaded into the data matrix.
